# E2E Tests

## Critical Flow Tested

User browsing catalog and adding a product to the cart.

Steps:
1. Open app
2. Log in like regular user
3. Browse catalog
4. Select a book
5. Add to cart
6. Verify cart has the book
7. Complete checkout

## Risks
- Frontend renders correctly
- Catalog loads 
- Cart functionality works 
- Checkout completed successfully
